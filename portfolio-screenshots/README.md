# Spectrum For Us — Captures portfolio Malt

Captures haute résolution (retina ×2) de la marketplace **Spectrum For Us**, générées
depuis l'app en local sur les vraies données du catalogue (Supabase).

> Note : les visuels produits sont des tuiles dégradées de remplacement (les photos
> produit sont hébergées sur Supabase Storage, non accessible depuis l'environnement
> de capture). Tout le reste — titres, prix, boutiques, catégories, textes — est réel.
